Discription of 5 HTML tags

<section> </section>
Indicates related content on the page. It should be used with heading tags.
<div> </div>
This tag allows grouping the area enclosed by div tag as a block-level element.
<q> </q>
This tag is used to cite short texts.
Ex - My grandma always says Playing is just as important as studying.
<strong> </strong>
This tag is used indicate text emphasis.
<img>
This is a singular tag for putting images on a web page.
It is used with src which specifies the path to the image, and alt which provides a description of the image.
